# AI Workbench Prototype

## 실행 방법

### 1) 프로젝트 폴더로 이동

```bash
cd /Users/yewon/capstoneFr
```

### 2) 패키지 설치

```bash
npm install
```

### 3) 개발 서버 실행

```bash
npm run dev
```

실행 후 터미널에 아래와 비슷한 메시지가 나오면 정상입니다.

```text
VITE v5.x.x ready in xxx ms
Local: http://localhost:7634/
```

브라우저에서 `http://localhost:7634/`로 접속하면 앱을 확인할 수 있습니다.

### 4) 서버 종료

개발 서버를 종료하려면 터미널에서 `Ctrl + C`를 누르세요.

## 추가 명령어

- 빌드: `npm run build`
- 빌드 결과 미리보기: `npm run preview`
